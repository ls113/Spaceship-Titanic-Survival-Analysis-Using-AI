📊 Model Description

This machine learning model predicts whether a passenger was transported to another dimension during the Spaceship Titanic incident. It is trained using structured passenger data such as age, spending behavior, home planet, and cabin details.

The Random Forest algorithm is used due to its high accuracy, robustness, and ability to handle missing values and complex relationships.

🧾 Features Used

  • Home Planet
  • Destination
  • Age
  • VIP Status
  • Room Service Spending
  • Food Court Spending
  • Shopping Mall Spending
  • Spa Usage
  • VR Deck Usage

📈 Performance Metrics

  • Accuracy: 85%
  • Precision: 83%
  • Recall: 82%
  • F1 Score: 82.5%

🔍 Key Insights

  • Passengers with high luxury spending are more likely to be transported.
  • VIP passengers show different survival patterns.
  • Cabin location plays a significant role.
